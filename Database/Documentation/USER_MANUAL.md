# User Manual

## Introduction

Public Budget Tracker helps citizens monitor government projects, budgets, complaints, and audit records.

---

## Dashboard

The dashboard displays:

- Total Projects
- Completed Projects
- Delayed Projects
- Over-Budget Projects
- Citizen Complaints

---

## Search Projects

Users can search projects using:

- Project ID
- Project Name
- District

---

## View Project Details

Each project displays:

- Budget Allocation
- Budget Spent
- Progress Percentage
- Status
- Complaint Count

---

## Chatbot Usage

Example Questions:

- Which projects are delayed?
- Show over-budget projects.
- What is the budget of Project P003?
- How many complaints exist for Project P005?

---

## Troubleshooting

### No Data Displayed

Refresh the page and verify the dataset is loaded.

### Invalid Search

Ensure the Project ID or Project Name is correct.
